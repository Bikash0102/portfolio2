My Interest

Web Development

MERN stands for MongoDB, Express.js, React, and Node.js, which are a group of technologies used for building web applications.
MongoDB is a NoSQL document-oriented database that is used to store application data.
Express.js is a back-end web framework for Node.js that is used to handle HTTP requests and provide APIs for the application.
React is a JavaScript library for building user interfaces, allowing developers to create reusable UI components.
Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine, and is used to execute server-side code.
By using these technologies together, a full-stack web application can be built, with a front-end built in React and a back-end built with Node.js and Express.js, backed by a MongoDB database. The combination of these technologies is referred to as the MERN stack.
